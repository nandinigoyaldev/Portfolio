# Portfolio Website

A static portfolio site organized into a clean, GitHub-friendly repository structure.

## Project Structure

- `index.html`: Main website entry point
- `assets/`: Images, scripts, icons, and fonts used by the page

## Local Development

Open directly:

```bash
xdg-open index.html
```

Or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Notes

- The active website files are at repository root (`index.html` + `assets/`).
