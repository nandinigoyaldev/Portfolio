# Contributing

Thanks for your interest in contributing to this portfolio repository!

## How to Contribute

1. **Fork** the repository.
2. Create a **new branch**:
   ```bash
   git checkout -b feature/your-change
   ```
3. Make changes and ensure the site still works by opening `index.html` in a browser.
4. Commit with a clear message:
   ```bash
   git commit -m "Add ..."
   ```
5. Open a **Pull Request**.

## Reporting Issues

If you find a bug or have an idea, please open an issue with:
- What happened
- Steps to reproduce (if applicable)
- Expected vs actual behavior

## Code Style

- Keep formatting consistent with existing HTML.
- Prefer small, focused changes.
